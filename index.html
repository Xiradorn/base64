Base64 Image Encoder Tool - Sir Xiradorn

Xir-Base64 Image Encoder Tool è uno strumento per la conversione di immagini nel formato base64. Questa codifica vi permetterà di usare la vostra immagine senza bisogno di caricarla da qualche parte e o di storarla nella cache del browser. Verrà letta come se fosse semplicissimo testo e ricostruita. Puo essere usata nel vostro html o anche il css. Attenzione : questa procedura è adatta solo a file dimensioni adeguate. Immagini troppo grandi potrebbero causare problemi di dimensionamento e non essere elaborate dal server . Inoltre potrebbero sovraccaricare la vostra cache del browser che non risponderebbe correttamente.

Qui sopra vi è stato mostrato qualche esempio di immagini rappresentate puramente con tale tecniche e che, ovviamente non sono state precaricare su questo o altri host di vario tipo.
